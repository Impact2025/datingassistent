## üé¨ VIDEO SLIDES

### Dimensies
- **Formaat:** 16:9
- **Resolutie:** 960 x 540px (HeyGen optimaal)
- **Export:** PNG voor achtergronden

### Layout Principes
1. **Witruimte:** Minimaal 30% van slide
2. **Tekst links:** Iris staat rechts
3. **Max 3 punten** per slide
4. **Accent bar:** 6-8px koraal lijn onder titels

### Slide Types
| Type | Layout |
|------|--------|
| Titel slide | Grote tekst gecentreerd |
| Content slide | Tekst links, ruimte rechts voor Iris |
| Kernpunten | Iconen + korte tekst, verticaal |
| Quiz slide | Vraag boven, opties onder |
