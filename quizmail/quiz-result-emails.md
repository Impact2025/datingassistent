## Implementatie in SendGrid/Code

### Variabelen die je nodig hebt:

```typescript
interface QuizEmailData {
  name: string;
  email: string;
  result_type: 'secure' | 'anxious' | 'avoidant' | 'fearful';
  result_title: string; // "De Stabiele Basis" etc.
  kickstart_link: string;
}
```

### Mapping result_type → result_title:

```typescript
const resultTitles = {
  secure: 'De Stabiele Basis',
  anxious: 'De Toegewijde Zoeker', 
  avoidant: 'De Onafhankelijke',
  fearful: 'De Paradox'
};
```

### SendGrid Template Approach:

**Optie A: 4 aparte templates**
- Maak 4 SendGrid templates (één per type)
- Select template based on `result_type`

**Optie B: 1 dynamic template met conditionals**
- Gebruik Handlebars in SendGrid
- `{{#equals result_type "anxious"}}...{{/equals}}`

**Aanbeveling:** Optie A is cleaner en makkelijker te onderhouden.
