# Iris Onboarding Videos

Upload your HeyGen videos here with the following filenames:

## Required Files

### Intro Video
- `iris-intro.mp4` - Main video (MP4 format, H.264)
- `iris-intro.webm` - Optional WebM for better compression
- `iris-intro-poster.jpg` - Thumbnail/poster image (first frame)
- `iris-intro.vtt` - Dutch captions (WebVTT format)

### Outro Video
- `iris-outro.mp4` - Main video (MP4 format, H.264)
- `iris-outro.webm` - Optional WebM for better compression
- `iris-outro-poster.jpg` - Thumbnail/poster image (first frame)
- `iris-outro.vtt` - Dutch captions (WebVTT format)

## Recommended Specs

| Setting | Value |
|---------|-------|
| Resolution | 1920x1080 (1080p) or 1280x720 (720p) |
| Aspect Ratio | 16:9 |
| Codec | H.264 (MP4) / VP9 (WebM) |
| Bitrate | 2-5 Mbps |
| Duration | 15-30 seconds |
| File Size | < 10MB per video |

## Caption Format (WebVTT)

Create `.vtt` files with this format:

```vtt
WEBVTT

00:00:00.000 --> 00:00:03.000
Hoi! Ik ben Iris, jouw persoonlijke dating coach.

00:00:03.500 --> 00:00:07.000
De komende minuten stel ik je een aantal vragen...
```

## Notes

- Videos will autoplay muted (browser policy)
- Users can tap to unmute
- Skip button appears after 2 seconds (intro) or immediately (outro)
- Captions are shown by default
